## 📌 Project Overview

A complete **end-to-end Data Analytics project** on IPL (Indian Premier League) Cricket data covering exploratory data analysis, data visualization, and sentiment analysis.

**Dataset:** [IPL Complete Dataset 2008–2020](https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020) — Kaggle

---

## ✅ Tasks Completed

| # | Task | Description |
|---|------|-------------|
| 1 | 📂 Data Collection | Loaded & inspected IPL match and ball-by-ball data |
| 2 | 🔍 Exploratory Data Analysis | Trends, patterns, correlations, hypothesis testing |
| 3 | 📊 Data Visualization | 8 charts + master dashboard using Matplotlib & Seaborn |
| 4 | 💬 Sentiment Analysis | Classified IPL news headlines using TextBlob & VADER |

---

## 🔍 Key Findings

- 🏆 **Virat Kohli** is the all-time leading run scorer in IPL history
- 🪙 Winning the toss gives **almost no advantage** — match outcomes are ~50/50
- ⚡ The best batsmen have **both** high averages AND high strike rates
- 💬 IPL news headlines are predominantly **positive or neutral** in sentiment
- 🎳 **Lasith Malinga** is among the all-time leading wicket takers

## 🚀 How to Run

**1. Clone the repository**
```bash
git clone https://github.com/YourUsername/CodeAlpha_IPLAnalytics.git
```

**2. Install dependencies**
```bash
pip install pandas numpy matplotlib seaborn textblob vaderSentiment requests beautifulsoup4
python -m textblob.download_corpora
```

**3. Open the notebook**
- Upload `IPL_Analytics_CodeAlpha.ipynb` to [Google Colab](https://colab.research.google.com)
- Upload `matches.csv` and `deliveries.csv` when prompted
- Run all cells from top to bottom ▶️
